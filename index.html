Projeto Letramento Digital Para Terceira Idade

Este projeto de extensão pretende capacitar os idosos com habilidades digitais essenciais para promover uma participação mais ativa nesta era digital, fortalecendo a conexão entre gerações.
O projeto é desenvolvido pela prof Iara Carnevale de Almeida - docente no curso de Sistemas para Internet, UniALFA Umuarama / PR - apoiado pelos alunos deste curso. 4 encontros presenciais

Período: Terças e Quintas-feiras às 9h-12h

Local: Biblioteca da UniALFA Umuarama
